Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Убедиться, что проект корректно собирается `Release` профилем
2. [ ] Создать пакет с помощью CPack `@[.agents/rules/cpp-pack.md]`
3. [ ] Включить все dynamic библиотеки в redistributable-дистрибутив
4. [ ] При необходимости сгенерировать Windows Installer (.exe)
5. [ ] Осуществить тестовую установку/распаковку на чистой системе
