description: Настройка локального CI/CD для C++
---
Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Инициализировать git-репозиторий и настроить корректный `.gitignore`
2. [ ] Прочитать требования к скрипту `setup_hooks.sh` из `@[.agents/rules/cpp-ci-cd.md]`
3. [ ] Настроить хук `pre-commit` для запуска `clang-format` только на staged файлах
4. [ ] Настроить хук `pre-push` для сборки через CMake и запуска CTest
5. [ ] Убедиться в наличии цветного вывода и установки прав `chmod +x` для хуков
6. [ ] Оформить скрипты как идемпотентные (повторный запуск безопасен)
