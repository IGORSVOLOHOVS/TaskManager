Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Включить флаги `--coverage` в свойстве компилятора CMake `@[.agents/rules/cpp-test-coverage.md]`
2. [ ] Собрать проект с тестовыми конфигурациями профилировщика
3. [ ] Выполнить полный прогон CTest для сбора сырых данных (`.gcda`, `.gcno`)
4. [ ] Сгенерировать HTML-отчет, используя инструменты `lcov` или `gcovr`
5. [ ] Проанализировать отчет, найти файлы с плохим покрытием
6. [ ] Сформировать рекомендации для дописывания тестов
