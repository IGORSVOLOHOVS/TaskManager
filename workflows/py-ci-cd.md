description: Локальный CI/CD для Python
---
Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Убедиться в наличии установленного Git и `pyproject.toml`
2. [ ] Использовать стандарты локального CI/CD `@[.agents/rules/py-ci-cd.md]`
3. [ ] Интегрировать `pre-commit` хук на автоматический запуск `ruff check` и `ruff format`
4. [ ] Подготовить `pre-push` хук для прогона всех тестов через `pytest`
5. [ ] Проверить работу хуков добавлением тестового коммита
