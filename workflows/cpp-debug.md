name: cpp-debug
description: Настройка отладки в VS Code
---
Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Создать конфигурационный файл `.vscode/tasks.json` на сборку (`cmake --build`)
2. [ ] Сконфигурировать `.vscode/launch.json` под нужный отладчик (GDB, LLDB или MSVC) `@[.agents/rules/cpp-debug.md]`
3. [ ] Указать корректные аргументы запуска и путь к бинарному файлу
4. [ ] Запустить тестовую сессию с Breakpoint для проверки `launch.json`
