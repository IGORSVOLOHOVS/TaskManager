Создай интерактивный Task List со следующими шагами и начни выполнение:
1. [ ] Создать структуру файлов тестирования (e.g. `test_*.py`) для модуля
2. [ ] Изучить руководство по написанию pytest-тестов `@[.agents/rules/py-test.md]`
3. [ ] Написать фикстуры (fixtures) для инициализации среды тест-кейса
4. [ ] Описать тестовые сценарии
5. [ ] Запустить `pytest` и убедиться в надежности выполнения
6. [ ] Дополнить отсутствующие тесты, если покрытие падает
